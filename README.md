## 📖 About the Project

This project showcases an **interactive Power BI Marketing Analytics Dashboard** designed to analyze marketing campaign performance using key business metrics such as **ROI, Clicks, Impressions, Conversion Rate, Customer Engagement, and Campaign Performance**.

Built using **Power BI Desktop**, **Power Query**, **DAX**, and **Microsoft Excel**, the dashboard transforms over **200,000 marketing campaign records** into meaningful business insights through interactive visualizations and KPI reporting.

The dashboard helps marketing teams monitor campaign effectiveness, compare marketing channels, evaluate customer engagement, and support data-driven decision-making.

# 🎯 Business Problem

Marketing teams often run multiple campaigns across different channels and target audiences. Tracking campaign performance, identifying high-performing marketing channels, and measuring return on investment becomes difficult without an interactive reporting solution.

This dashboard simplifies marketing performance analysis by providing a centralized view of important KPIs and visual insights.

# 📂 Dataset

**Dataset Name:** Marketing Campaign Performance Dataset

**Source:** Kaggle

**Dataset Link**

https://www.kaggle.com/datasets/manishabhatt22/marketing-campaign-performance-dataset

The dataset contains over **200,000 marketing campaign records**, including:

- Campaign ID
- Company
- Campaign Type
- Marketing Channel
- Target Audience
- Acquisition Cost
- ROI
- Clicks
- Impressions
- Conversion Rate
- Engagement Score
- Customer Segment
- Location
- Language
- Duration
- Date

> **Note:** The dataset is not included in this repository because it exceeds GitHub's upload size limit. You can download it directly from Kaggle.
